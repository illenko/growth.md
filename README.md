## Quick Start

### Installation

#### From Source

```bash
# Clone the repository
git clone https://github.com/illenko/growth.md.git
cd growth.md

# Build the binary
make build

# Install to $GOPATH/bin (optional)
make install
```

### Usage

```bash
# Initialize a new growth repository
growth init ~/my-growth

# Create your first skill
growth skill create "Python" --category programming --level intermediate

# Create a goal
growth goal create "Become ML Engineer" --priority high

# Generate AI-powered learning path (coming soon)
growth path generate goal-001

# Log your progress
growth progress log "Completed Fast.ai Lesson 1"

# View your overview
growth overview
```
