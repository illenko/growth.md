## Core Concepts

### Skills
Individual competencies you're developing (e.g., "Machine Learning", "Distributed Systems", "System Design")

### Goals  
High-level career objectives (e.g., "Become ML Engineer", "Master Backend Architecture")

### Learning Paths
AI-generated, personalized sequences of skills and resources tailored to your goals and current level

### Milestones
Significant achievements in your journey (e.g., "First ML model in production", "Led system design review")

### Resources
Curated learning materials (books, courses, articles, videos) linked to specific skills

### Progress
Daily/weekly logs tracking what you've learned and accomplished

## CLI Commands (Draft)

```bash
# Initialize
growth init "My Engineering Career"

# Skills
growth skill create "Machine Learning"
growth skill list [--level beginner|intermediate|advanced]
growth skill view ml
growth skill edit ml --level intermediate

# Goals
growth goal create "Become ML Engineer" --skills ml,python,math
growth goal list [--status active|completed|archived]
growth goal view ml-engineer

# Learning Paths (AI-generated)
growth path generate --goal ml-engineer
growth path view ml-engineer
growth path update ml-engineer  # Re-generate based on progress

# Resources
growth resource add "Book" "Deep Learning" --skill ml --url amazon.com/...
growth resource list [--skill ml]
growth resource search "transformers"

# Progress
growth progress log "Completed Fast.ai Lesson 1"
growth progress log --milestone ml-model-1 "Deployed first model"
growth progress view [--week] [--month]
growth progress stats

# Milestones
growth milestone create "First ML Model in Production"
growth milestone complete ml-prod-1
growth milestone list

# Board/Overview
growth board           # TUI showing current focus areas
growth overview        # Statistics and progress summary
growth browser         # Web UI (like backlog.md)

# Search
growth search "neural networks"  # Search across all files
```

## AI Integration (MCP)

Like backlog.md's MCP integration, growth.md works seamlessly with AI assistants:

```bash
# Setup (similar to backlog.md)
growth init  # Offers MCP setup
claude mcp add growth --scope user -- growth mcp start
```

**AI Assistant Capabilities:**
- "Claude, generate a learning path to become an ML engineer"
- "What should I learn next based on my current skills?"
- "Find resources about distributed systems"
- "Log my progress: completed chapter 3 of DDIA"
- "Show me similar engineers' paths" (if using shared paths)

## FAQ

**Q: How is this different from backlog.md?**  
A: backlog.md manages project tasks; growth.md manages career development. Different domain, similar philosophy.

**Q: Do I need to use Git?**  
A: Yes, growth.md is git-native. But it handles git operations for you.

**Q: Does the AI require API keys?**  
A: For AI features, you'll need OpenAI/Anthropic API key. But basic tracking works without AI.

**Q: Can I share my learning path?**  
A: Yes! Just push your growth/ folder to GitHub. Others can fork your path.

**Q: Will there be a hosted version?**  
A: No plans for MVP. It's local-first by design. But community might build sync services.

**Q: Can teams use this?**  
A: V1 is individual-focused. Team features might come in V2+ based on demand.
